# SMART Budget - Selective Source Code Disclosure Preview

This repository contains the security-critical components of SMART Budget, published for independent verification of our privacy claims.

## Objective
Enable experts to verify that:
- Privacy is enforced by client-side encryption.
- No PII is sent to AI models.
- Keys never leave the user's device.

## Scope
