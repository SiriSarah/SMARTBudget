# SMART Budget Public Disclosure Repository

## About SMART Budget

SMART Budget is a modern, privacy-first budgeting application designed to give users complete control over their financial data. Built as a Progressive Web App (PWA), it operates entirely offline by default, ensuring your financial information stays on your device. Key features include:

- **Envelope Budgeting**: Zero-based budgeting with rollover support
- **Multi-Currency Support**: Track transactions in 50+ currencies with live exchange rates
- **AI-Powered Insights**: Optional AI advisor with comprehensive PII sanitization
- **Local-First Architecture**: All data encrypted locally using AES-256-GCM
- **Cross-Platform**: Works on desktop, mobile, and tablets
- **11 Language Support**: Including RTL languages and Indic scripts

## Why This Public Repository?

In the spirit of transparency and to build trust with our users, we're making key components of our application publicly available for independent review and verification. This repository contains our security whitepaper, privacy policy, and the source code for our core cryptographic and sanitization modules. By open-sourcing these critical components, we allow security researchers, developers, and users to verify our claims about data privacy and security.
